<h1 align="center">Dark-Mode</h1>

## Project Description

React dark-mode toggler using imported toggler and dates using moment.js

## Built with

- HTML
- CSS
- React

# npm-start

[npm start](https://docs.npmjs.com/cli/v7/commands/npm-start)

```
npm install
npm install react-icons --save
```

```
npm-start -g

```
